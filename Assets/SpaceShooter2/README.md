# SpaceShooter2

Отдельная учебная игра для модуля 6.

## Структура

- `Scenes` - сюда можно сохранить сцену `SpaceShooter.unity`
- `Prefabs` - корабль, пули, астероиды, враги, UI
- `Sprites` - спрайты корабля, астероидов, врагов, фона
- `Audio` - звуки выстрела, попадания и поражения
- `UI` - визуальные элементы интерфейса
- `Docs` - разбор по урокам и чеклист сборки
- `Scripts` - код игры и поэтапные версии по урокам

## Идея модуля

Модуль 6 лучше строить как простой `2D` космический шутер:

- урок 1 должен быть коротким и понятным;
- к концу урока уже есть движение и выстрел;
- основная механика заканчивается к уроку 3;
- урок 4 остаётся в основном под визуал, звук, баланс и билд.

## Ключевые темы

- движение по осям
- выстрел и время жизни объекта
- `Invoke` и спавнеры
- инкапсуляция
- `public`, `private`, `[SerializeField]`
- атрибуты Unity
- наследование из ООП
- `GameManager`
- очки и состояние матча
- собственный `UnityEvent`
- полировка и билд

## Папки с результатом по урокам

- `Scripts/Lesson1` - движение и первый выстрел
- `Scripts/Lesson2` - `Invoke`, астероиды, враги, спавнеры
- `Scripts/Lesson3` - `GameManager`, очки, конец игры, свой `UnityEvent`
- `Scripts/Lesson4` - финальная версия модуля с минимумом нового кода
