# TowerDefenseRPG

Учебная игра для модуля 5.

## Идея проекта

Игрок защищает точку обороны с помощью арбалета:

- стреляет по врагам;
- получает валюту за победу;
- покупает улучшения;
- переживает несколько волн.

## Почему эта игра подходит под модуль 5

- на старте можно сделать очень маленький результат: первый выстрел;
- дальше удобно разносить роли по объектам:
  - арбалет;
  - стрела;
  - враг;
  - защищаемая точка;
  - менеджер игры;
- `GameManager`, улучшения и волны естественно появляются позже;
- 6 урок можно оставить почти без нового кода и посвятить полировке.

## Структура

- `Docs` - описание уроков, сборки сцены и настроек игры
- `Scripts/Lesson1` - код на конец 1 урока
- `Scripts/Lesson2` - код на конец 2 урока
- `Scripts/Lesson3` - код на конец 3 урока
- `Scripts/Lesson4` - код на конец 4 урока
- `Scripts/Lesson5` - код на конец 5 урока
- `Scripts/Lesson6` - код на конец 6 урока

## Логика по урокам

- `M5У1` - проект, сцена, арбалет, первый выстрел
- `M5У2` - классы и объекты: враг, стрела, точка защиты
- `M5У3` - методы с `return`: `CanShoot()`, `IsAlive()`, `HasLost()`
- `M5У4` - `GameManager`, валюта, улучшения
- `M5У5` - волны врагов, разные префабы врагов
- `M5У6` - минимум нового кода, баланс, визуал, звук, билд

## UnityEvent

После базового ООП сюда удобно добавить отдельную маленькую тему:

- свой `UnityEvent` для смерти врага;
- свой `UnityEvent<int>` для награды;
- событие обновления UI без жёсткой привязки.

Это лучше давать как расширение после основной логики модуля, а не как тяжёлую обязательную тему в 6 уроке.
