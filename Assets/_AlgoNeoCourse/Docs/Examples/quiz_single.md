# Пример Quiz — Single Choice

Минимальный пример вопроса с одним правильным ответом.

## Что важно

- используйте `kind: single`;
- указывайте уникальный `id`;
- должен быть ровно один вариант с `correct: true`;
- ответ фиксируется сразу по клику;
- состояние квиза сохраняется локально автоматически.

Настройки доступны в:

```text
Tools -> AlgoNeoCourse -> Settings -> Open Quiz Settings
```

## Пример

```quiz
id: sc-hello
kind: single
text: Какой модификатор доступа делает поле приватным в C#?
answers:
  - text: private
    correct: true
  - text: public
  - text: protected
  - text: internal
```

Обычно следующий слайд содержит пояснение.

---

## Пояснение (пример)

- `private` ограничивает доступ полем текущего класса;
- `public` открывает доступ всем;
- `protected` даёт доступ наследникам;
- `internal` ограничивает доступ сборкой, а не классом.
