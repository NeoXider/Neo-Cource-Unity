# Пример: открыть ассет из урока

Откройте префаб игрока:

[Открыть префаб](unity://open?path=Assets/Prefabs/Player.prefab)

Также поддерживаются проектные пути и поиск по имени:
```
[Открыть материал](unity://open?path=Assets/Materials/Player.mat)
```

Советы:
- Для относительных ссылок используйте `Assets/...` или `Packages/...`.
- При ошибке пути в консоль выводится сообщение, урок продолжает работу.
